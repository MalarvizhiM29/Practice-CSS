CSS Comments/Colors/Width& Height


CSS Comments

To comment in CSS, simply place your plain text inside /* */ marks.


CSS Colors

Css colours can be defined using three types,
1)color name
2)Hexadecimal-value must start with hash
3)RGB Colour value
Hexadecimal and RCB color values are taken from the colorpick extension.


CSS Height and Width

The CSS height and width properties are used to set the height and width of an element.px and % symbols are used to define the height and width.
